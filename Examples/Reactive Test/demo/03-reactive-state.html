Automatic UI Updates

With reactive state, your UI automatically updates when data changes. No manual DOM manipulation required!

1. Simple Reactive Counter

The display updates automatically when the counter state changes.
Count: 0
const state = ReactiveUtils.state({ count: 0 }); Conditions.whenState( () => state.count, { '0': { textContent: 'Count: 0', style: { color: 'gray' } }, '>0': { textContent: `Count: ${state.count}`, style: { color: 'green' } }, '<0': { textContent: `Count: ${state.count}`, style: { color: 'red' } } }, '#counter-output' ); // Changes update automatically! state.count++;

2. Loading Status Indicator

Shows different states: loading, success, or error with automatic updates.
Ready
const state = ReactiveUtils.state({ status: 'idle' }); Conditions.whenState( () => state.status, { 'loading': { textContent: '⏳ Loading...', style: { color: '#2196f3', background: '#e3f2fd' } }, 'success': { textContent: '✓ Success!', style: { color: '#4caf50', background: '#f1f8f4' } }, 'error': { textContent: '✗ Error occurred', style: { color: '#f44336', background: '#fef1f0' } } }, '#status-output' );

3. Reactive Form Validation

Real-time validation feedback as user types.
Enter an email to validate
const state = ReactiveUtils.state({ email: '' }); Conditions.whenState( () => state.email, { 'empty': { textContent: 'Enter an email to validate' }, '/^[^@]+@[^@]+\\.[^@]+$/': { textContent: '✓ Valid email', style: { color: '#4caf50', fontWeight: 'bold' } }, 'truthy': { textContent: '✗ Invalid email format', style: { color: '#f44336', fontWeight: 'bold' } } }, '#validation-output' );

4. Reactive Theme Switcher

Switch themes with automatic UI updates across multiple elements.
Current theme: light
const state = ReactiveUtils.state({ theme: 'light' }); Conditions.whenState( () => state.theme, { 'light': { textContent: '☀️ Light Theme', style: { background: '#fff', color: '#333' } }, 'dark': { textContent: '🌙 Dark Theme', style: { background: '#1e1e1e', color: '#fff' } } }, '#theme-display' ); // Automatically updates all bound elements! state.theme = 'dark';

5. Reactive Shopping Cart

Add items and watch the cart update automatically with conditional styling.
Cart is empty
const cart = ReactiveUtils.state({ items: 0 }); Conditions.whenState( () => cart.items, { '0': { textContent: 'Cart is empty', style: { color: 'gray' } }, '1-5': { textContent: `${cart.items} item(s) in cart`, style: { color: 'blue' } }, '>5': { textContent: `🛒 ${cart.items} items - Ready to checkout!`, style: { color: 'green', fontWeight: 'bold' } } }, '#cart-output' );
