Demo — .visible()

Filters the collection to only elements that are not hidden via display:none, visibility:hidden, or opacity:0. Returns an array. Matched cards get a cyan border — unmatched cards dim.

Interactive — toggle visibility, then run filter

Use the hide buttons to hide individual cards in different ways, then click Run .visible() to see which ones the filter returns.
