Demo — Selector.configure(options)

Merges the provided options object into the Selector helper's internal settings using Object.assign. Common options: maxCacheSize, enableLogging, enableEnhancedSyntax, cleanupInterval, enableSmartCaching. Returns Selector for chaining.

vs Collections.configure(): Both accept the same pattern and return their global object for chaining. Selector adds Selector-specific options like enableSmartCaching and cleanupInterval. Unknown keys are silently ignored.
Important: configure({ enableEnhancedSyntax: true }) only saves the option — it does NOT activate the Proxy. Call Selector.enableEnhancedSyntax() explicitly to wrap query/queryAll with a Proxy.
